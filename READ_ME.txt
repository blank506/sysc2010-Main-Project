!. To use our GUI first press the load CSV button at the top of the window. This will open the "All_Data_Sets_Testing" folder. Our project is designed to operate with "MLTempDataset.csv" for temperature, "processedData_Subject05_FEM.csv" for respiration, and "s10_run.csv" and "s10_walk.csv" for ECG.

2. Once your file has been selected the raw time domain signal will be displayed for the first 10s.

3. In the dropdown below you must select the type of data used. See the first line.

4. Next FIR or IIR can be selected for the type of FFT desired.

5. Pressing the "Low Pass", "High Pass", or "Band pass" buttons will display a time domain plot of the original signal as well as the corresponding filtered counterpart.

6. "Show FFT" will display the FFT of the original signal as well as the most recently selected filter type. If a signal has not yet been selected only the original will be shown.

7. The  "Reset" button brings the system back to 2.

8. Below the buttons you will see the extracted statistical metrics and other features based on the type of signal selected. Note that these values are updated only once a filter type has been selected and the plot has been closed.
